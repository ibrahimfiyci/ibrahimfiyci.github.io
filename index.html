Security work

James Fiyci is focused on practical security work: designing cloud security architectures, studying Zero Trust controls, improving detection engineering, documenting incident response workflows, and testing how AI systems can be deployed with safer guardrails.

This site is a place for cybersecurity studies: clear projects, defensive research notes, cloud and AI architecture patterns, security engineering practice, and readable risk communication.
